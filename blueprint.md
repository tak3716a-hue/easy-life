# Project Blueprint

## Overview

This is a simple web application that includes a Lotto Number Generator and a contact form for affiliation inquiries.

## Features

### Lotto Number Generator
*   Generates 6 unique random numbers between 1 and 45.
*   Displays the generated numbers to the user.

### Affiliation Form
*   Provides a simple form to send partnership inquiries.
*   The form includes fields for name, email, and a message.
*   Submissions are handled by Formspree.
